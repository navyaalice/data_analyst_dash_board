## Call Center Performance Dashboard – Power BI
## Project Overview

This project presents an interactive Call Center Performance Dashboard built using Power BI. The dashboard analyzes operational and customer service KPIs to provide actionable business insights for improving performance and customer satisfaction.

The dataset was provided as part of a virtual internship task and is used for learning and portfolio purposes.

## Objectives

Monitor key call center KPIs
Evaluate agent performance
Analyze customer satisfaction metrics
Identify peak call hours and workload patterns
Generate insights for operational improvement

## Key Metrics Analyzed

Total Calls
First Call Resolution (FCR)
Customer Satisfaction Score (CSAT)
Customer Effort Score (CES)
Average Handling Time (AHT)
Answered vs Resolved Calls
Call Type Distribution

## Dashboard Features

KPI Cards for quick performance overview

Agent performance comparison (bar charts)

Call volume trend analysis (line charts)

Call type breakdown (pie chart)

Resolution distribution (donut charts)

Agent-level detailed performance table

## Tools & Skills Used

Power BI
DAX Measures
Data Modeling
Data Cleaning & Transformation
Business Intelligence & KPI Reporting

## Business Insights

Identified performance gaps between agents.
Highlighted peak call hours for workforce planning.
Analyzed customer satisfaction trends.
Suggested improvements to increase FCR and reduce AHT.
